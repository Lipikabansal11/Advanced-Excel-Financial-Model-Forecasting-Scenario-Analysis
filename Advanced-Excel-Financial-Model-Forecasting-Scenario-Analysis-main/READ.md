# Apex Solutions Financial Modelling

A comprehensive **Excel-based financial modelling project** developed to analyze the financial performance of **Apex Solutions**, forecast future financial statements, and estimate the company's intrinsic value using industry-standard valuation techniques.

## Project Overview

This project demonstrates the complete process of building an integrated **3-Statement Financial Model** in Microsoft Excel. Historical financial data is linked with forecasting assumptions to generate projected financial statements, perform business valuation, and analyze multiple business scenarios.

## Features

- Built an integrated **Income Statement, Balance Sheet, and Cash Flow Statement**.
- Forecasted revenue, operating expenses, depreciation, taxes, and capital expenditure.
- Developed debt and working capital schedules for dynamic financial forecasting.
- Performed **Discounted Cash Flow (DCF)** valuation using projected Free Cash Flows.
- Conducted **Sensitivity Analysis** using different WACC and terminal growth assumptions.
- Built **Scenario Analysis** for Best, Base, and Worst Case business projections.
- Evaluated business performance using profitability, liquidity, and solvency ratios.
